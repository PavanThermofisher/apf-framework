## How to Deploy (Free on GitHub Pages)

### Step 1: Create a New Repository

1. Go to GitHub.com and sign in with your account (dpavansrinivas@gmail.com)
2. Click the **+** icon in the top right corner
3. Select **New repository**
4. Repository name: **apf-framework** (or any name you prefer)
5. Description: "APF Framework - Agents, Processes, Frameworks for AI-Augmented Development"
6. Select **Public**
7. Check **Add a README file**
8. Click **Create repository**

### Step 2: Upload Your Website Files

1. In your new repository, click **Add file** → **Upload files**
2. Drag and drop ALL these files:
   - index.html
   - about.html
   - resources.html
   - contact.html
   - styles.css
3. Add commit message: "Initial website launch"
4. Click **Commit changes**

### Step 3: Enable GitHub Pages

1. In your repository, click **Settings** (top menu)
2. Scroll down and click **Pages** (left sidebar)
3. Under "Source", select **main** branch
4. Click **Save**
5. Wait 1-2 minutes for deployment

### Step 4: Access Your Live Website

Your website will be live at:
```
https://[your-github-username].github.io/apf-framework/
```

For example, if your username is "pavansrinivas", it will be:
```
https://pavansrinivas.github.io/apf-framework/
```

## Updating Your Website

To make changes later:

1. Go to your repository
2. Click on the file you want to edit
3. Click the pencil icon (Edit)
4. Make changes
5. Scroll down and click **Commit changes**
6. Changes appear live in 1-2 minutes
